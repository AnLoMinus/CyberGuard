## ⚙️ **script.js**

```javascript
// Light hover sparks effect
document.querySelectorAll(".card").forEach(card => {
    card.addEventListener("mousemove", e => {
        card.style.boxShadow = `0 0 25px #00cfffaa`;
    });
    card.addEventListener("mouseleave", () => {
        card.style.boxShadow = "none";
    });
});
```
